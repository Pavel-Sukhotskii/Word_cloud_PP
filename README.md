# Word_cloud_PP

Function inputs:
* logs - table
* column_ids:list - list of question id's like ['QOp0a', 'QOp0b']
* translation=None - translate from. Example: 'italian'
* rows=False - 'True' if questions and answers in rows (Will be updated)
