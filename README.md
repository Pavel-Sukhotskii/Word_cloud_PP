# Word_cloud_PP

Function inputs:
* <u>logs</u> - table
* <u>column_ids:list</u> - list of question id's like ['QOp0a', 'QOp0b']
* <u>translation=None</u> - translate from. Example: 'italian'
* <u>rows=False</u> - 'True' if questions and answers in rows (Will be updated)
